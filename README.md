How to add it to your browser (Note: It doesn't work in windows web browser and the mac broswer is mistry too, it works fine in dabian web browser but it should be chromium base web browser.)
1st, download the zip/cole the project to your pc or laptop 
2nd, extract that zip(move to step 3 if you clone the git repo) 
3rd, open your browser and cick on extension icon on your right side of search bar then mannage extension
4th, enable developer mode to the right top of the extension manager page 
5th, click on load unpack then selecet the downloaded (extracted folder or clone repo)
6th, you are good to go, select what sound effect you want use it for your liking.
